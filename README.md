# 🎧 Audiobook Sales Analysis & Purchase Prediction
A machine learning project that analyzes audiobook sales data and predicts whether a user will purchase an audiobook using a neural network.

## 🔍 What This Project Does
- Analyzes customer behavior from audiobook sales data
- Handles class imbalance and scales features
- Trains a neural network to predict purchase decisions
- Evaluates performance using accuracy and a confusion matrix

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- TensorFlow (Keras)
- Matplotlib

## 🤖 Model
- Neural Network with two hidden layers (ReLU)
- Output layer for binary classification
- Optimized using Adam with early stopping

## 📊 Evaluation
- Training vs validation loss and accuracy
- Confusion matrix on test data

## 🚀 How to Run
- download the audiobook_data.csv from repository
- Open the notebook and run all cells from top to bottom.

## 📌 Outcome
The model successfully predicts whether a user is likely to buy an audiobook based on historical data.
